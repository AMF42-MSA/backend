## JPA에서 특정 컬럼 제거
1. @Entity등록된 프로그램에서 로깅을 위하여 logger필드를 추가하면 오류 발생(static, final 필드 사용으로 오류)
    ```java
        private final Logger log = LoggerFactory.getLogger(LectureController.class);
    ```

2. 특정 필드 예외 처리 방법
    ```java
        @Transient  //  Hibernate will ignore it
        private final Logger log = LoggerFactory.getLogger(LectureController.class);
    ```
